# 🧠 Natural Language Processing (NLP) Projects

A collection of beginner-to-intermediate NLP projects developed using Python. This repository demonstrates various Natural Language Processing techniques including text translation, text-to-speech conversion, sentiment analysis, text summarization, resume screening, mental health sentiment analysis, and news classification.

---

# 📂 Projects Included

## 🌐 Language Translator & Text-to-Speech

### Description
Translates English text into multiple Indian languages and converts the translated text into speech.

### Features
- English to Malayalam, Hindi, Tamil, Telugu, and Kannada translation
- Text-to-Speech (TTS) conversion
- Automatic audio file generation

### Technologies Used
- Google Translate API (`googletrans`)
- gTTS
- Python

---

## 😊 Sentiment Analysis System

### Description
Analyzes user reviews and determines sentiment polarity.

### Features
- Positive, Negative, and Neutral sentiment detection
- Polarity and Subjectivity scores
- Word and Character count
- Most frequent word identification
- CSV report generation

### Technologies Used
- TextBlob
- Pandas
- Collections

---

## 📝 Text Summarization

### Description
Generates concise summaries from long paragraphs using extractive summarization.

### Features
- Automatic text summarization
- Two-sentence summary generation

### Technologies Used
- Sumy
- LSA Summarizer

---

## 📄 Resume Screening System

### Description
Matches resumes with job descriptions using NLP techniques.

### Features
- Resume preprocessing using SpaCy
- TF-IDF Vectorization
- Cosine Similarity Matching
- Candidate Ranking System

### Technologies Used
- SpaCy
- Scikit-learn
- TF-IDF
- Cosine Similarity

---

## 💙 Mental Health Sentiment Analyzer

### Description
Analyzes user text to identify mood and emotional indicators.

### Features
- Mood Classification (Positive, Neutral, Negative)
- Stress and Anxiety Keyword Detection
- Mood Score Calculation
- Mood History Tracking
- CSV Data Storage

### Technologies Used
- TextBlob
- Pandas
- Datetime

---

## 📰 News Category Classification

### Description
Classifies news articles into predefined categories using Machine Learning.

### Features
- TF-IDF Text Vectorization
- Logistic Regression Model
- Model Accuracy Evaluation
- Live News Prediction

### Technologies Used
- Scikit-learn
- Pandas
- Logistic Regression
- TF-IDF
